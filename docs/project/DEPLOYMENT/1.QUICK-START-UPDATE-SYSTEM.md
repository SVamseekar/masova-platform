## Prerequisites (One-Time Setup)

### 1. Docker Hub Account (FREE)

```bash
# Sign up at: https://hub.docker.com
# - Create account
# - Verify email
# - Create repository: masova (public, free)
```

### 2. GitHub Repository

```bash
# If not already done:
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/masova.git
git push -u origin main
```

### 3. Add Secrets to GitHub

```bash
# Go to: GitHub Repo → Settings → Secrets and variables → Actions
# Add these secrets:

DOCKER_USERNAME    # Your Docker Hub username
DOCKER_PASSWORD    # Your Docker Hub password or access token
```

**Get Docker Access Token:**
1. Docker Hub → Account Settings → Security → New Access Token
2. Copy token
3. Paste in GitHub secret

## Troubleshooting

### Build fails with "Maven not found"

```bash
# Install Maven
brew install maven  # Mac
apt install maven   # Ubuntu
# Or download from: https://maven.apache.org
```

### Docker Hub push fails with "unauthorized"

```bash
# Check login
docker login

# Check username in script
# Edit build-monolith.sh line:
DOCKER_IMAGE="yourusername/backend"  # Use YOUR username
```

### GitHub Actions fails

```bash
# Check secrets
# GitHub → Settings → Secrets
# Make sure DOCKER_USERNAME and DOCKER_PASSWORD are set

# Check workflow logs
# GitHub → Actions → Click on failed workflow
# Read error message
```

### UpdateService doesn't detect new version

```bash
# Check version.properties on GitHub
curl https://raw.githubusercontent.com/yourusername/masova/main/version.properties

# Should return:
# version=1.0.1
# buildDate=2025-12-26T...
# releaseUrl=...

# Check UpdateService configuration
# api-gateway/src/main/resources/application.yml
# update.check.enabled: true
```
